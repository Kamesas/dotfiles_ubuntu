# Hyprland Keybindings - Omarchy Style

## Essential Apps
- `Super + Q` - Open terminal (kitty)
- `Super + Space` - Application launcher (rofi)
- `Super + E` - File manager

## Window Management
- `Super + W` - Close active window
- `Super + F` - Fullscreen
- `Super + Shift + V` - Toggle floating mode
- `Super + M` - Exit Hyprland
- `Super + P` - Pseudo tile (dwindle)
- `Super + J` - Toggle split direction

## Focus Navigation
- `Super + Arrow Keys` - Move focus between windows
- `Alt + Tab` - Cycle to next window
- `Alt + Shift + Tab` - Cycle to previous window

## Window Positioning
- `Super + Shift + Arrow Keys` - Swap window with neighbor
- `Super + Mouse Left Drag` - Move window
- `Super + Mouse Right Drag` - Resize window

## Workspaces
- `Super + [1-9,0]` - Switch to workspace 1-10
- `Super + Shift + [1-9,0]` - Move window to workspace 1-10
- `Super + Tab` - Next workspace
- `Super + Shift + Tab` - Previous workspace
- `Super + Mouse Scroll` - Cycle through workspaces

## Scratchpad
- `Super + S` - Toggle scratchpad
- `Super + Shift + S` - Move window to scratchpad

## Screenshots
- `Print` - Select area and copy to clipboard
- `Shift + Print` - Select area and save to ~/Pictures
- `Super + Print` - Full screen to clipboard

## Media Keys
- `XF86AudioRaiseVolume` - Volume up 5%
- `XF86AudioLowerVolume` - Volume down 5%
- `XF86AudioMute` - Toggle mute
- `XF86MonBrightnessUp` - Brightness up
- `XF86MonBrightnessDown` - Brightness down
- `XF86AudioNext` - Next track (requires playerctl)
- `XF86AudioPlay/Pause` - Play/pause (requires playerctl)
- `XF86AudioPrev` - Previous track (requires playerctl)

## Tips
- Your wallpaper is at ~/.config/hypr/wallpaper.png - replace it with your own!
- Screen locks after 10 minutes of inactivity
- Notifications appear in top-right corner
- Status bar (waybar) shows workspaces, window title, time, and system info
